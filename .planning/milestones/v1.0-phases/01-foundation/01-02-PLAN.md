    ```typescript
    import type { DiffEntry } from "./types";
    import { TuningSection } from "./types";
    import { FIELD_METADATA } from "./fieldMetadata";
    ```

    1. `computeDiff(current: Record<string, unknown>, preset: Record<string, unknown>, prefix?: string): DiffEntry[]`
       - Recursively walk `preset` keys.
       - For each leaf value (non-object, or array), build the dot-path: `prefix ? prefix + "." + key : key`.
       - If the dot-path exists in `FIELD_METADATA` AND the value differs from `current[path]`:
         - Create a DiffEntry with path, label from FIELD_METADATA, section from FIELD_METADATA, oldValue from current, newValue from preset.
       - If the value is a non-null object (and not an array), recurse with updated prefix.
       - Skip paths NOT in FIELD_METADATA (this naturally excludes definition registries per research anti-pattern).
       - To read nested values from `current`, use a helper that walks the dot-path:
         ```typescript
         function getNestedValue(obj: Record<string, unknown>, path: string): unknown {
           return path.split(".").reduce<unknown>(
             (acc, key) => (acc && typeof acc === "object" ? (acc as Record<string, unknown>)[key] : undefined),
             obj,
           );
         }
         ```

    2. `groupDiffBySection(entries: DiffEntry[]): Record<string, DiffEntry[]>`
       - Group entries by their `section` property.
       - Initialize with all 4 TuningSection values mapped to empty arrays.
       - Return the grouped object.
