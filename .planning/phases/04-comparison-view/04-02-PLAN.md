Create `creator/src/components/tuning/MetricCard.tsx`:

This component renders a single KPI card for one TuningSection. It receives `currentMetrics` and `presetMetrics` (both MetricSnapshot) plus a `section` prop (TuningSection), and displays 2-3 curated metric rows with before/after values and delta badges.

**Component structure:**

```typescript
import { useRef, useEffect, useMemo } from "react";
import tippy from "tippy.js";
import { TuningSection } from "@/lib/tuning/types";
import type { MetricSnapshot } from "@/lib/tuning/types";
import { pctDelta, deltaDirection, deltaColor } from "@/lib/tuning/deltaUtils";

interface MetricRowData {
  label: string;
  current: number;
  preset: number;
  format: (n: number) => string;
  formulaTooltip: string;
}

interface MetricCardProps {
  section: TuningSection;
  currentMetrics: MetricSnapshot;
  presetMetrics: MetricSnapshot;
  diffCount?: number;
}
```

**Curated metrics per section (per D-04, using Normal/"standard" tier per D-03, at Lv10/Lv30/Lv50 per D-02):**

Combat & Stats card -- 3 metrics:
- "Mob HP -- Lv10": `mobHp["standard"][10]`, formula tooltip: "baseHp + hpPerLevel x level (Normal tier)"
- "Mob HP -- Lv30": `mobHp["standard"][30]`, formula tooltip: "baseHp + hpPerLevel x level (Normal tier)"
- "Dodge Chance": `dodgeChance[10]`, format as `${(n*100).toFixed(1)}%` if fraction or `${n.toFixed(1)}%` if already percentage. Formula tooltip: "agility stat bonus / (agility stat bonus + 100)"

Note on dodgeChance: check the actual values in MetricSnapshot. The `dodgeChance` function returns `Math.min(bindings.dodgePerPoint * dodgeStatValue, bindings.maxDodgePercent)` -- this is already a percentage number (e.g., 5.0 meaning 5%). Format with `${n.toFixed(1)}%`.

Progression & Quests card -- 3 metrics:
- "XP to Lv10": `xpPerLevel[10]`, format with comma separator, formula tooltip: "Cumulative XP from level 1 to N using the XP exponent curve"
- "XP to Lv30": `xpPerLevel[30]`, format with comma separator, formula tooltip: "Cumulative XP from level 1 to N using the XP exponent curve"
- "Player HP -- Lv10": `playerHp[10]`, format as integer, formula tooltip: "baseHp + hpPerLevel x level"

Economy & Crafting card -- 2 metrics + footnote:
- "Gold/Kill -- Lv10": `mobGoldAvg["standard"][10]`, format as number with 1 decimal, formula tooltip: "mob base gold x goldMultiplier (Normal tier)"
- "Gold/Kill -- Lv30": `mobGoldAvg["standard"][30]`, format as number with 1 decimal, formula tooltip: "mob base gold x goldMultiplier (Normal tier)"
- Footnote text: "Buy/sell multipliers visible in raw fields below" in `text-text-muted text-[12px]`

World & Social card -- 1 metric + changed-count note:
- "Regen Interval": `regenInterval[1]`, format as `${n.toFixed(0)}ms`, formula tooltip: "regenIntervalBase (milliseconds per regen tick)"
- If `diffCount` prop is provided and > 0, show: `"{diffCount} other fields changed"` in `text-text-muted text-[12px]`

**MetricRow rendering (internal function, not exported):**

Each row is a flex container with:
- Label (Crimson Pro 14px, text-text-secondary) with a Tippy tooltip on hover showing the formulaTooltip string (D-12). Use vanilla tippy via useRef + useEffect, theme "arcanum", placement "top-start", delay [200, 100], maxWidth 260.
- "Current" column header: first row gets a small "Current" / "Preset" header above values.
- Current value (JetBrains Mono 13px, text-text-muted, right-aligned)
- Preset value + delta (JetBrains Mono 13px, color from deltaColor(deltaDirection(current, preset)), right-aligned). Format: `{formattedPresetValue} {pctDelta(current, preset)}` when direction is not "same", or `{formattedPresetValue} —` when same.

- Surface: `bg-bg-secondary rounded-xl p-4 border border-border-muted`

const sectionDiffCounts = useMemo(() => {
  if (!selectedPresetId || !config) return undefined;
  const preset = TUNING_PRESETS.find((p) => p.id === selectedPresetId);
  if (!preset) return undefined;

    config as unknown as Record<string, unknown>,
    preset.config as unknown as Record<string, unknown>,
  );
  const grouped = groupDiffBySection(diffs);
  const counts = new Map<string, number>();

    counts.set(section, entries.length);

  return counts;
