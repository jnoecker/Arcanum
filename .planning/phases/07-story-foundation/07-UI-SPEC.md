## Color

| Role | Token / Value | Usage |
|------|---------------|-------|
| Dominant (60%) | `bg-bg-primary` (#2a3149) | Page background, panel interiors |
| Secondary (30%) | `panel-surface` class (gradient `--bg-panel`) | Cards, dialog bodies, StoryEditorPanel surface |
| Accent (10%) | `--color-warm` (#c8a46a) / `action-button-primary` | Primary CTA buttons ("Create Story"), cover image actions |
| Story Template Color | `--color-template-story: #c98fb8` (NEW) | Story dot color in ArticleTree, story icon tint |
| Destructive | `action-button-danger` / `--color-status-danger` (#dbb8b8) | Delete Story button, destructive confirmations |

Accent reserved for:
- "Create Story" primary CTA button in NewStoryDialog
- "New Story" toolbar button in ArticleTree (if primary)
- Cover image "Choose Image" / "Change Image" action button
- Active/selected story article highlight in ArticleTree

NOT accent: form labels, section headers, dividers, metadata text. Those use `text-text-primary` or `text-text-secondary`.

### New CSS Token Required

```css
/* Add to @theme block in creator/src/index.css */
--color-template-story: #c98fb8;
```

This rose-mauve color is visually distinct from all 11 existing template colors. It sits between `--color-template-ability` (#b88faa) and the cooler tones, providing clear differentiation.

Source: `creator/src/index.css` lines 108-119 (existing template colors), RESEARCH.md code example

## NewStoryDialog -- Layout Contract

**Trigger:** "New Story" button in ArticleTree toolbar (alongside existing "New Article" button).

**Shell:** `DialogShell` with `widthClassName="max-w-lg"`, title "Create Story".

**Form fields (top to bottom, 16px vertical gap between fields):**

1. **Title** -- Text input, required. Placeholder: "The Dark Awakening". Label: "Story Title". Auto-focused on dialog open.
2. **Zone** -- Dropdown/select, required. Label: "Linked Zone". Options populated from loaded zones (`zoneStore`). Placeholder: "Select a zone". If no zones loaded, show disabled state with helper text "Load a zone first".
3. **Cover Image** -- Optional. Thumbnail preview (64x64px rounded-lg) + "Choose Image" action button (secondary variant). If no image selected, show a placeholder rectangle with dashed border and text "No cover image". Uses existing `AssetPickerModal` for selection.

**Footer (right-aligned, 8px gap between buttons):**
- Never Mind: `ActionButton` variant="ghost"
- Create Story: `ActionButton` variant="primary", disabled until title and zone are filled

**Scrim:** `DialogShell` default scrim (dark overlay with backdrop-blur)

---

## StoryEditorPanel -- Layout Contract

**Context:** Rendered inside `ArticleBrowser` when `selectedArticle.template === "story"`. Replaces `ArticleEditor` for story articles.

**Container:** `panel-surface` class, `min-h-[32rem] rounded-3xl p-5` (matches existing ArticleEditor container).

**Layout (vertical stack, 24px gaps between sections):**

### Section 1: Header Bar
- Story title as `font-display text-lg text-text-primary` (editable inline or via pencil icon)
- Zone badge: small pill showing linked zone name, `text-2xs font-display uppercase tracking-[0.5px] text-text-muted bg-bg-elevated rounded-full px-3 py-1`
- Undo/Redo buttons: top-right, `ActionButton` variant="ghost" size="icon", disabled when stack is empty

### Section 2: Cover Image
- If cover image set: 240px wide thumbnail with rounded-xl border, `border border-border-default`. Hover shows "Change" overlay.
- If no cover image: 240px x 160px dashed-border placeholder rectangle. Center-aligned text: "Add a cover image" in `text-text-muted text-sm`. Click opens `AssetPickerModal`.

### Section 3: Scene List Placeholder
- Empty state centered in remaining space
- Ornate divider (`ornate-divider` class)
- Heading: "No scenes yet" (`font-display text-lg text-text-primary`)
- Body: "Scenes will be added in a future update. For now, your story is saved and ready." (`text-sm text-text-muted leading-7 max-w-sm`)
- This section is a placeholder for Phase 8 (timeline editor)

### Section 4: Metadata Footer
- Created/updated timestamps in `text-2xs text-text-muted`
- Story ID in `font-mono text-2xs text-text-muted` (for debugging, subtle)

## Interaction States

### NewStoryDialog

| State | Behavior |
|-------|----------|
| Open | Focus trap active, title input auto-focused, Escape closes |
| Title empty | "Create Story" button disabled (opacity 0.45, cursor not-allowed) |
| Zone not selected | "Create Story" button disabled |
| Both filled | "Create Story" button enabled (warm gold accent border + glow on hover) |
| Submitting | "Create Story" button shows Spinner, all inputs disabled |
| Success | Dialog closes, new story article selected in tree, StoryEditorPanel opens |
| Error | Inline error text below the form in `text-status-error text-sm` |

### StoryEditorPanel

| State | Behavior |
|-------|----------|
| Loading story | Centered Spinner in panel |
| Story loaded | Metadata sections render, scene placeholder shown |
| Story not found | Error state: "This story's data could not be loaded." with `text-status-error` |
| Cover image hover | Subtle overlay with "Change" text appears over thumbnail |
| Dirty (unsaved) | Title in sidebar shows dirty indicator (existing lore dirty pattern) |
| Undo available | Undo button enabled (full opacity) |
| Undo unavailable | Undo button disabled (opacity 0.45) |

### ArticleTree -- Story Articles

| State | Behavior |
|-------|----------|
| Default | Film icon in rose-mauve (#c98fb8), title in `text-text-primary` |
| Selected | Background highlight (existing tree selection style) |
| Hover | Background `bg-bg-hover` (existing tree hover style) |
| Filtered by "story" template | Only story articles visible (existing template filter) |

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Dialog focus trap | `useFocusTrap` hook (existing, used by `RewriteDialog`) |
| Dialog escape to close | Keyboard handler on `DialogShell` (built-in) |
| Form labels | `<label htmlFor>` on all inputs in NewStoryDialog |
| Button disabled states | `aria-disabled` + visual opacity change |
| Screen reader announcement | `role="dialog"` + `aria-labelledby` on DialogShell (built-in) |
| Story icon alt text | `aria-label="Story"` on the film SVG icon in ArticleTree |
| Color contrast | All text meets WCAG AA against `bg-bg-primary` (verified: #dbe3f8 on #2a3149 = 8.2:1 ratio) |

## Animation

| Element | Animation | Timing | Easing |
|---------|-----------|--------|--------|
| DialogShell open | Fade in + scale from 0.95 to 1.0 | 200ms | `--ease-unfurl` (cubic-bezier(0.16, 1, 0.3, 1)) |
| DialogShell close | Fade out + scale to 0.95 | 150ms | `--ease-dissolve` (cubic-bezier(0.7, 0, 0.84, 0)) |
