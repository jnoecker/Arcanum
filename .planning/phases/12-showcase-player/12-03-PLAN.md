**IntersectionObserver for animation triggers (D-12):**
- Track which scenes have been visited: `const [visitedScenes, setVisitedScenes] = useState<Set<string>>(new Set());`
- Track which scene is currently in view: `const [activeSceneId, setActiveSceneId] = useState<string | null>(story.scenes[0]?.id ?? null);`
- `useEffect` sets up an IntersectionObserver with `threshold: 0.5` and `root: containerRef.current`.
- Observer callback: when a scene entry `isIntersecting`:
  - Set `activeSceneId` to that scene's ID.
  - If the scene ID is NOT in `visitedScenes`, add it (it's a newly seen scene, will play entrance animations).
- Scene `playing` prop: `playing={activeSceneId === scene.id && !visitedScenes.has(scene.id)}` -- only plays entrance animations for scenes being seen for the first time.
- After entrance animations complete (via a small setTimeout matching animation duration ~1500ms), add scene ID to `visitedScenes`. This ensures:
  - Scrolling forward to a new scene: animations play.
  - Scrolling back to a previously visited scene: shows final state (no re-animation), per D-12.

Use a two-phase system:
1. `playingScenes` state set: scenes that are currently animating.
2. `completedRef` ref set: scenes that have finished animating.
3. When IntersectionObserver fires for a new scene:
   - If NOT in `completedRef`: add to `playingScenes` (triggers `playing=true`).
   - If in `completedRef`: do nothing (shows final state).
4. After ~2 seconds (generous buffer for all animations), move from `playingScenes` to `completedRef`.

Implementation:
```typescript
const [playingScenes, setPlayingScenes] = useState<Set<string>>(new Set());
const completedRef = useRef<Set<string>>(new Set());

// IntersectionObserver callback
const handleIntersection = useCallback((entries: IntersectionObserverEntry[]) => {
  for (const entry of entries) {
    if (entry.isIntersecting) {
      const sceneId = entry.target.getAttribute("data-scene-id");
      if (sceneId && !completedRef.current.has(sceneId)) {
        setPlayingScenes(prev => new Set(prev).add(sceneId));
        // Mark as completed after animations finish
        setTimeout(() => {
          completedRef.current.add(sceneId);
          setPlayingScenes(prev => {
            const next = new Set(prev);
            next.delete(sceneId);
            return next;
          });
        }, 2000);
      }
      setActiveSceneId(sceneId);
    }
  }
}, []);
```
